# Dark Theme Update - Modern AI Security Blog

## New Color Scheme

### Background Colors
- **Sidebar:** `rgba(20, 18, 50, 0.90)` - Deep dark purple with high opacity
- **Post Cards:** `rgba(30, 27, 75, 0.75)` - Slightly lighter dark purple
- **Footer/Paginator:** Same as post cards for consistency

### Text Colors
- **Primary Text:** `#e5e7eb` - Light gray (excellent readability on dark)
- **Secondary Text:** `#9ca3af` - Medium gray for less emphasis
- **Links:** `#a78bfa` - Light purple (matches gradient theme)
- **Link Hover:** Underline with same purple

### Accent Colors
- **Borders:** `rgba(139, 92, 246, 0.3)` - Purple with transparency
- **Hover Borders:** `rgba(139, 92, 246, 0.5)` - Brighter purple on hover
- **Backgrounds (badges, code):** `rgba(139, 92, 246, 0.2)` - Subtle purple tint

### Effects
- **Backdrop Blur:** `blur(20px)` - Stronger blur for depth
- **Shadows:** Darker, more prominent shadows
- **Transitions:** Smooth 0.2-0.3s ease

## Visual Changes

### Before (White Theme)
```
Sidebar:    White (rgba(255, 255, 255, 0.88))
Post Cards: White (rgba(255, 255, 255, 0.92))
Text:       Dark gray (#1f2937)
Links:      Blue (#2563eb)
```

### After (Dark Theme)
```
Sidebar:    Dark purple (rgba(20, 18, 50, 0.90))
Post Cards: Dark purple (rgba(30, 27, 75, 0.75))
Text:       Light gray (#e5e7eb)
Links:      Light purple (#a78bfa)
```

## Design Philosophy

### Cohesive Flow
- Dark elements blend naturally with purple gradient background
- No harsh white contrasts
- Smooth visual transitions

### Modern Aesthetic
- Glassmorphism with stronger blur
- Layered depth with shadows
- Purple accent color throughout

### Readability
- High contrast text on dark backgrounds
- Light purple links stand out
- Clear visual hierarchy
